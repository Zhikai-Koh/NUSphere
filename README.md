# NUSphere: Proof of Concept

The following section delineates the successful implementation of core marketplace features, specifically focusing on desktop and mobile interfaces for category browsing, verified shops, and the integrated shopping cart ecosystem.

## 🗂️ Browsing by Categories (Desktop UI)
The application implements a streamlined, categorized browsing interface tailored for web users.

* Category Grid: The homepage features a dedicated "Categories" section with visual cards representing broad product groupings, such as "Electronics" and "Clothing," enabling users to immediately filter the marketplace.
* Centralized Search: A prominent search bar is integrated into the header beneath the primary NUSphere branding, allowing users to query specific items directly.
* Global Navigation: The header includes persistent icons for the user profile, the shopping cart, and notifications, ensuring these core tools remain accessible during browsing.

## 🛒 Integrated Cart & Verified Shops (Mobile UI)
The mobile-responsive layout effectively highlights storefront feeds and the persistent cart system.

* Verified Storefront Feed: The "Shops" tab renders a dynamic list of student-run businesses (e.g., 3D Printing Services, Gelato, Keyboard Mods). Each listing displays the shop name, a brief description, the seller's avatar, their aggregated rating, and a direct "View Shop" call-to-action.
* Persistent Shopping Cart: The top navigation retains the global shopping cart icon alongside the user profile and a notification bell (featuring an active unread badge), ensuring users can always track their intended purchases without losing their place in the shop feed.
* Quick-Access Bottom Navigation: A sticky bottom navigation bar allows users to seamlessly toggle between the "Open Market," the "Shops" directory, and a central "+" button to easily "Add Listing," fulfilling the project's goal of lowering the barrier to entry for student sellers.
